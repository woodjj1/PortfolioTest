Project Goals

Setting up 

 

  1. Laravel 5.8 - Setting up laravel was more challenging than any other general installation because it required us as a class to go through certain steps to get things working as it should.   After installing we needed to configure our .env file to allow laravel to work. 
  2. Composer- We then installed composer which is a PHP dependency manager.  This installs Laravel dependencies and lets us create a Laravel project. 
  3. Mamp - Using mamp (local web server)  allowed my laravel project to be viewed like a live website.  There were a few routine steps to setup mamp to work everytime we used the polytech computers.

Understanding Laravel

Going over the tutorial a couple more times after class gave me great satisfaction when everything worked.  Our lecturers were really great at making things clear and pointing out things we shouldn't do in the tutorial at the time but researching it will give better understanding for the future.
